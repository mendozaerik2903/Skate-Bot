# Skate Bot
Welcome to my mobile application project, "Skate Bot." This app is designed to provide skateboarders a way to practice and test their skateboarding skills by simulating typical skate games. This includes two modes: the classic SKATE game and a trick generator via skate dice. Users may customize the difficulty to their preference for each game and they can either play alone (and against a bot) or find other skateboarders to play with using the in-app map feature.
